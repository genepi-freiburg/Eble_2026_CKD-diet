# Causal Effects of Dietary Components on Kidney Function

**Two-Sample Mendelian Randomization with IEU OpenGWAS Instruments and Automated API-Based PheWAS Confounder Screening**

---

## Overview

Two-sample Mendelian Randomization (MR) study examining the causal effects of 10 dietary
exposures on kidney function (eGFR, CKD risk). Genetic instruments are extracted from the
IEU OpenGWAS platform via API and screened for pleiotropic confounding using automated
PheWAS and Steiger filtering. Replication is performed in FinnGen R12.

**Key finding:** Tea intake is the only FDR-significant dietary exposure with both eGFR
(β = +0.0162, P_FDR = 0.0002) and CKD risk (OR = 0.669, P_FDR = 0.0008), replicated in
FinnGen R12 (OR = 0.666, P_FDR = 0.042). All other exposures are null after FDR correction.

This repository is a minimal, stand-alone release containing the code and the primary
result tables needed to reproduce the main findings. Manuscript and supplementary tables,
publication figures, and intermediate sensitivity outputs are not included; they are fully
regenerated by the pipeline (see below).

## Reproducing the analysis

### 1. Prerequisites

**R ≥ 4.3** with the following packages:

```r
install.packages(
  c("ieugwasr", "MVMR", "dplyr", "tidyr", "data.table",
    "ggplot2", "patchwork", "scales"),
  repos = c("https://mrcieu.r-universe.dev", "https://cloud.r-project.org")
)
```

Python ≥ 3.9 (only for figure regeneration): `pandas`, `numpy`, `matplotlib`, `scipy`.

### 2. IEU OpenGWAS authentication

```bash
# Option A — environment variable (recommended)
export OPENGWAS_JWT="your_token"

# Option B — .env file
cp .env.example .env
# edit .env, paste your token, then:
set -a && source .env && set +a
```

Get a free token at https://api.opengwas.io/ (institutional email required; tokens are
valid ≈ 14 days).

> Never commit your token. `.env` is in `.gitignore`.

### 3. Download GWAS outcome summary statistics (~1.1 GB total)

```bash
bash scripts/01_download_gwas_data.sh
```

This downloads:

| File | Source | Size |
|---|---|---|
| `data/raw/egfr_EA.txt.gz` | CKDGen, Wuttke 2019 (eGFR, EA) | ~184 MB |
| `data/raw/ckd_EA.txt.gz`  | CKDGen, Wuttke 2019 (CKD, EA) | ~171 MB |
| `data/raw/finngen/finngen_R12_N14_CHRONKIDNEYDIS.gz` | FinnGen R12 | ~778 MB |

Manual sources if the script fails:

- CKDGen: https://ckdgen.imbi.uni-freiburg.de/
- FinnGen R12 manifest: https://storage.googleapis.com/finngen-public-data-r12/summary_stats/finngen_R12_manifest.tsv
- FinnGen R12 CKD direct: https://storage.googleapis.com/finngen-public-data-r12/summary_stats/release/finngen_R12_N14_CHRONKIDNEYDIS.gz

### 4. Run the analysis

```bash
Rscript analysis/01_run_mr_analysis.R
```

The full pipeline writes all canonical outputs to `data/processed/` and runs in
~5–10 min (assuming the OpenGWAS API is responsive).

### 5. Regenerate figures (optional)

```bash
python scripts/build_fig1.py              # Extended forest plot
python scripts/build_figS5_scatter_ckd.py # Scatter plots (CKD)
python scripts/build_figS5_mvmr.py        # MVMR forest plot
python scripts/build_graphical_abstract.py
```

## Study design

| Parameter | Details |
|---|---|
| Design | Two-sample MR (STROBE-MR compliant) |
| Exposures | 10 dietary traits (Tea, Fresh fruit, Dried fruit, Cooked veg, Salad, Oily fish, Processed meat, Beef, Pork, Lamb/mutton) |
| Exposure GWAS | UK Biobank via IEU OpenGWAS (Elsworth 2018; Cole 2020) |
| Instrument selection | P < 5×10⁻⁸, LD R² < 0.001, 10,000 kb window, EUR |
| PheWAS screening | Round 1: TG, BMI, T2D, SBP (P < 1×10⁻⁵); Round 2 (post-hoc): CAD — rs429358 (*APOE* locus) |
| Steiger filtering | 3 pairs removed: rs6059844 (Oily fish), rs6484504 (Processed meat), rs11211124 (Pork) |
| Final instruments | **264 harmonised SNP–exposure pairs** (10–55 per exposure) |
| Primary outcomes | eGFR (log-transformed) + CKD (binary), CKDGen (Wuttke 2019, N = 480,698) |
| Replication | FinnGen R12 CKD `N14_CHRONKIDNEYDIS` (N_total = 493,235; 12,787 cases) |
| MR methods | IVW, Weighted Median, MR-Egger, Weighted Mode; MVMR (6 exposures); LOO |
| Sensitivity analyses | eGFRcys (Pattaro 2016, `ieu-a-1106`, N = 33,152 — no UK Biobank overlap); GI-PRAL leave-one-out across all 10 foods |

## Methodological notes

### Instrument pipeline

1. **Extraction** — `tophits()` via IEU API (P < 5×10⁻⁸, R² < 0.001, 10,000 kb, EUR): **326 pairs**
2. **PheWAS Round 1** — 4 confounder GWAS (TG, BMI, T2D, SBP), P < 1×10⁻⁵
3. **PheWAS Round 2** — CAD post-hoc (rs429358, *APOE* locus)
4. **Harmonisation** — rsID inner join + allele alignment (palindromic SNPs with MAF in [0.42, 0.58] dropped)
5. **Steiger filter** — `r2_exposure` > `r2_outcome` per pair: **264 pairs final**

### F-statistics

All instruments satisfy F > 25. The per-SNP F-statistic is F = β²/SE² (= z²), computed
from the UK Biobank exposure GWAS summary statistics. The per-exposure mean/min/max F
reported in the manuscript is the arithmetic mean of these **univariable** per-SNP
F-statistics over the **final retained** instrument set, written to
`data/processed/table1_instrument_strength.csv`. This is distinct from the
Sanderson–Windmeijer **conditional** F reported for the MVMR model
(`data/processed/mvmr_conditional_fstats.csv`).

### MVMR

MVMR-IVW estimates are numerically identical to univariable IVW for all 6 exposures,
because each combined instrument is associated with exactly one exposure (zero SNP
overlap), indicating independent biological pathways. Conditional F-statistics
(Sanderson–Windmeijer): Fresh fruit 12.94 and Oily fish 14.56 are above the conventional
threshold of 10; Processed meat, Cooked veg, Salad, and Beef are below. MVMR results are
therefore presented as a supplementary sensitivity rather than primary.

### GI-PRAL composite

GI-PRAL = Σᵢ (wᵢ × β̂_IVW,ᵢ) where wᵢ are per-serving PRAL weights from Remer & Manz 1995,
*J Am Diet Assoc* 95:791–797 (Table 2). Hypothesis-generating only — the signal is driven
by Pork; leave-one-out without Pork is non-significant.

### eGFRcys sensitivity (Pattaro 2016)

A cystatin C-based eGFR outcome (`ieu-a-1106`, N = 33,152, no UK Biobank overlap) is used
as an alternative biomarker robust to muscle-mass confounding. Outcome statistics are
pulled live via `ieugwasr::associations(id = "ieu-a-1106")`, harmonised with the
Steiger-filtered instrument set, and IVW-tested per exposure. No exposure reaches nominal
significance, consistent with the smaller sample size and the borderline nature of all
non-Tea exposures.
